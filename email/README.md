# Instruções para envio de e-mail utilizando o Spring Boot Mail

## Incluir a dependência no pom.xml

```bash
  <dependency>
		<groupId>org.springframework.boot</groupId>
		<artifactId>spring-boot-starter-mail</artifactId>
	</dependency>
```
